# 🧹 Duplicate Cleaner — 智能文件去重整理工具\n\n> 一键扫描重复文件，智能整理归类，还你一个干净的硬盘！\n\n![License](https://img.shields.io/badge/license-MIT-blue)\n![Python](https://img.shields.io/badge/python-3.8%2B-green)\n\n## ✨ 功能\n\n- 🔍 **SHA256 精确去重** — 连一个字节都不放过\n- 📝 **文件名模糊匹配** — 相同内容不同文件名也能识别\n- 📁 **自动归类整理** — 图片、文档、视频、音频、压缩包自动分类\n- 📊 **详细报告** — 删除/移动了什么，节省了多少空间\n- 🗑️ **安全模式** — 先预览再操作，误删可恢复\n- 🎨 **彩色终端输出** — 好看易读\n\n## 🚀 快速开始\n\n### 🐍 Python 版（推荐）\n\\\ash\npython cleaner.py --scan .                  # 扫描预览\npython cleaner.py --scan . --dedupe          # 扫描+去重\npython cleaner.py --scan . --organize        # 扫描+整理\npython cleaner.py --scan . --dedupe --organize --report  # 全流程\n\\\\n\n### 💚 Node.js 版（无Python环境可用）\n\\\ash\nnode cleaner.js --scan .                    # 扫描预览\nnode cleaner.js --scan . --dedupe            # 扫描+去重\nnode cleaner.js --scan . --organize          # 扫描+整理\nnode cleaner.js --scan . --dedupe --organize --report  # 全流程\n\\\\n\n## 🎯 使用场景\n\n- **下载目录** — 重复下载的安装包、文档\n- **照片目录** — 导出多次产生的重复照片\n- **工作目录** — 版本混乱的多份备份\n- **桌面** — 杂乱无章的桌面文件\n\n## 📦 输出示例\n\n\\\\n╔══════════════════════════════════════════════╗\n║   🧹 Duplicate Cleaner - 扫描报告             ║\n╚══════════════════════════════════════════════╝\n\n📁 扫描目录: C:\Users\Downloads\n📊 总文件数: 1,234\n🔁 重复文件: 89 组 (共 312 个副本)\n💾 可节省: 2.4 GB\n\n📂 文件归类统计:\n  📷 图片: 342 个 (1.2 GB)\n  📄 文档: 156 个 (89 MB)\n  🎵 音频: 23 个 (456 MB)\n  🎬 视频: 12 个 (3.1 GB)\n  📦 压缩包: 45 个 (890 MB)\n  ❓ 其他: 56 个 (234 MB)\n\n✨ 操作完成！已删除 89 个重复文件，节省 1.2 GB\n\\\\n\n## 💖 支持这个项目\n\n如果这个工具帮到了你，欢迎微信扫码打赏一杯咖啡 ☕\n\n![微信收款码](assets/wechat_qr.jpg)\n\n你也可以给我 ⭐ Star 以示鼓励！\n\n---\n\n> **变干净吧！** 🧹\n
