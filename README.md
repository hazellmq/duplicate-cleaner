# 🧹 Duplicate Cleaner — 智能文件去重整理工具

> 一键扫描重复文件，智能整理归类，还你一个干净的硬盘！

![License](https://img.shields.io/badge/license-MIT-blue)
![Python](https://img.shields.io/badge/python-3.8%2B-green)

## ✨ 功能

- 🔍 **SHA256 精确去重** — 连一个字节都不放过
- 📝 **文件名模糊匹配** — 相同内容不同文件名也能识别
- 📁 **自动归类整理** — 图片、文档、视频、音频、压缩包自动分类
- 📊 **详细报告** — 删除/移动了什么，节省了多少空间
- 🗑️ **安全模式** — 先预览再操作，误删可恢复
- 🎨 **彩色终端输出** — 好看易读

## 🚀 快速开始

```bash
# 安装
pip install -r requirements.txt

# 扫描当前目录的重复文件（仅预览，不动文件）
python cleaner.py --scan .

# 扫描并自动整理文件到分类文件夹
python cleaner.py --scan . --organize

# 预览重复文件，手动选择要删除的副本
python cleaner.py --scan . --dedupe

# 完整模式：去重 + 整理 + 报告
python cleaner.py --scan . --dedupe --organize --report
```

## 🎯 使用场景

- **下载目录** — 重复下载的安装包、文档
- **照片目录** — 导出多次产生的重复照片
- **工作目录** — 版本混乱的多份备份
- **桌面** — 杂乱无章的桌面文件
